## Summary
The kitchen receives orders from customers.

## Tech Stack:
- Java 17;
- PostgreSQL 15;
- Spring Boot 2;
- Hibernate 5;
- Lombok 1;
- Liquibase 4;

## Environment
JDK 17, IntelliJ IDEA CE 2021.3.3, Maven 3.8.1, PostrgreSQL 15

## Launch
1. Create database: ```kitchen```.
2. Run the application from the Main class.
3. Start Postman and run requests.
